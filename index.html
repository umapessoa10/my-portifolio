PokeYou

Projeto em makecode arcade.

Este projeto foi realizado nas aulas de tecnologia para explorar as capacidades de UI e diferentes sistemas no aplicativo. O jogo foi profundamente inspirado em Pokémon, contando com algumas semelhanças como o sistema de batalhas em turno e um sistema de mochilas e HP, mas tendo como diferença que você batalha ao invés de seus Pokemón

Raze

Projeto em HTML, CSS e JS

Este projeto foi feito em HTML, CSS e JS como forma de conhecer mais a língua e fazer uma sátira ao site Blaze, um site de apostas online que já foi acusado de farsa, acusações essas já provadas inúmeras vezes, por isso o site replica sua estética com textos que satirizam isso, como "Seja roubado em até R$600"
